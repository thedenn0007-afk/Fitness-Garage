IMAGES FOLDER — Fitness Garage Website
======================================

Place the following files in this folder:

1. logo.png
   - Your gym logo (transparent background preferred)
   - Then open index.html and follow the instructions in the two
     <!-- LOGO --> comment blocks to swap the text logo for your image.

2. 1.jpg  ← Used as the HERO background (full screen)
3. 2.jpg  ← Used in the About section + Gallery
4. 3.jpg  ← Used in the Gallery

If an image is missing, the site automatically falls back to a stock
Unsplash gym photo — so the site always looks good even before you
add the real photos.

Recommended image sizes:
  - 1.jpg (hero)   : 1920 × 1080 px minimum, landscape
  - 2.jpg (about)  : 800 × 1000 px, portrait preferred
  - 3.jpg (gallery): any size, will be cropped to fit
  - logo.png       : height ~80px, transparent background
